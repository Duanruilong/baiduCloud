## 名称修改

- 如果是放到 每一个Dom上面 添加一个修改键值 那就需要 放一个修改箭头
 
> 我更加倾向于第二种情况,第二种情况 必须要考虑数组的多少点击那个修改名称 就修改。
> 修改名称 需要都注意的是,如果是修改文件的名称,应该只修改文件名称部分不能修改

>在刚开始产生的时候  因该是1 显示名称 还有 修改键位

 并且将value值传给dom键值,并且显示编辑键,那么编辑健的作用及时 换类名
